# GPTECH - Interactive E-Commerce Platform
Project Overview:
GPTECH is a modern, high-performance e-commerce frontend built entirely with Vanilla JavaScript, HTML, and CSS. Designed to mimic the premium feel of top-tier tech brands, the project demonstrates advanced DOM manipulation, state management, and modern UI/UX principles without relying on heavy frameworks like React or Angular.

Key Features:

Custom SPA Router: Navigating between Home, Shop, Reviews, and User Profile happens instantly without the browser reloading.

Persistent Cart System: Users can add/remove items to their cart, with total prices calculated dynamically. The cart state is saved in localStorage, meaning users don't lose their items if they leave or refresh the page.

Live Search & Filtering: The shop features a real-time search bar and category filters that instantly update the product grid using JavaScript array filtering.

Interactive 3D Hero Section: A custom-built 3D rotating product cube that automatically spins but can also be manually controlled by the user.

Premium UI/UX: Features a glass-morphism sticky navigation bar, a functional Dark/Light theme toggle that remembers user preferences, and scroll-triggered animations powered by the Intersection Observer API.

Tech Stack:

HTML5 (Semantic Structure)

CSS3 (Variables, Flexbox, Grid, 3D Transforms, Keyframe Animations)

Vanilla JavaScript (ES6+, DOM Manipulation, LocalStorage API, Intersection Observer API)
